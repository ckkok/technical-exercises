# JavaScript QA Engineer

This page contains description and requirements for a QA Automation Engineer assignment.

### Duration

The duration of this assignment is generally two days.

## Goal
The goal of this assignment is for you to demonstrate your ability to:

* Design and implement automated test using Selenium
* Plan different test scenarios
* Write clean, reviewable code
* Use git
* Write clear and concise documentation

## Requirements

1. Create a public git repository that you will send to us.
2. Write an automated test suite for http://book.theautomatedtester.co.uk

It has a rich set of web UI functions specially designed for the needs of testing of web application of all types.
Some of the operations of the web elements that you should test are:

* Checkbox and radio button related
* Load and assert the dynamic text
* Launching of windows on click
* Ajax operations
* Dropdown and alerts
* [BONUS] Cookie related operations

3. Document how the tests can be run. Your documentation should include:

* Required tools to be installed prior to running the tests
* Steps for running the tests


## Tools

You may use the following tools, but are not limited to them:
* [Webdriverio](https://webdriver.io/)
* [Selenium](https://www.seleniumhq.org/)
